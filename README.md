# ⚡ SENTIMENT_OS v2.0
> **Neural Language Processing Interface for Real-Time Sentiment Analysis**

SENTIMENT_OS is a high-performance, deep learning-powered web interface designed to decode the emotional subtext of digital communication. Built with a custom BiLSTM architecture and a glassmorphism UI, it provides instant insights into the "vibe" of any text stream.

---

## 🚀 Key Features
* **Neural Scan Engine**: Powered by a custom-trained BiLSTM model for high-accuracy classification.
* **Cyberpunk UI**: A premium, dark-mode dashboard with real-time progress tracking and glassmorphism effects.
* **Signal History**: Persistent session logging to track sentiment shifts over time.
* **Low Latency**: Optimized inference delivery for near-instant results (~12ms).

## 🛠️ Tech Stack
* **Frontend**: Streamlit (Custom CSS/HTML Injection)
* **Intelligence**: TensorFlow / Keras (BiLSTM)
* **Processing**: Python, Pandas, NumPy
* **Deployment**: Streamlit Cloud / GitHub
